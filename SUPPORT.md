# Support

- Installation and operations: [`docs/SELF_HOSTING.md`](docs/SELF_HOSTING.md), [`docs/UPGRADE.md`](docs/UPGRADE.md)
- Compatibility: [`docs/COMPATIBILITY.md`](docs/COMPATIBILITY.md)
- Security vulnerabilities: report privately as described in [`SECURITY.md`](SECURITY.md)
- Reproducible non-security bugs: use the GitHub Issue Forms

Include exact `vX.Y.Z`/digest, OS and architecture, deployment method, relevant time window, expected/actual behavior, and minimal redacted logs. Never paste tokens, complete install commands, Authorization headers, `.env`, database/backup contents, notification credentials, or private keys. This community project provides best-effort support; the latest supported stable version is the first troubleshooting baseline.
